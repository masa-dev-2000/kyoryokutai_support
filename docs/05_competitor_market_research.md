## 🎯 エグゼクティブ・サマリ

1. **大手SI(富士通・NEC・NTTデータ等)は構造的に参入不可**

3. **真の脅威は水平SaaS(gamba!・Leafwork 等)の参入(2年以内)**
4. **令和7年4月の随契基準引上げ(委託200万円まで)が追い風**

## 2. kintone の技術制約(堀の根拠)

### 仮説「単一テナント」の検証結果: **正しい**
- kintone は 1契約=1サブドメイン(`xxx.cybozu.com`)
- 別自治体間のデータ共有は以下の迂回路のみ、いずれも構造的に横断ナレッジ集約に不向き:
    - **ゲストスペース**: 招待者は1スペースに閉じ込め、他スペース不可、モバイル不可
    - **REST API / DataSyncer**: 外部同期サービス、追加コスト・工数が必要
    - **組織間アクセス権**: 同一ドメイン内限定
- 「複数自治体でデータ横断」を実装した公的事例は**発見できず**

### サイボウズ「シェアDX」の正体
- 「自治体間でアプリテンプレを共有」する取り組み
- 代表例: コロナワクチン抽選(加古川→岐阜)、避難所運営(別府)
- **重要**: テンプレ共有 ≠ データ共有。各自治体で個別運用
- **提案時注意**: 「テンプレ共有で十分」と顧客が誤認するリスクあり

## 3. 競合分析

### 3.1 大手SI(富士通・NEC・NTTデータ・TIS・日立)

**不参入の構造的理由**:
- 最小案件規模: **3,000万〜億円/年**(推測)
- 人月単価: NEC 205万、NTTデータ 178万、富士通 同水準
- NECガバクラ運用支援の最低: **月額80万円**(約960万円/年)
- 協力隊市場単価 20〜100万円 × 1自治体 では赤字確定

**参戦リソース状況**:
- 標準化17業務(2025末期限)で完全埋まり、スケジュール通り3割だけ(ダイヤモンド報道)
- その後もガバクラ移行の長い尾

**協力隊特化の公表提案・採択事例: ゼロ**

**評価: 窓は5年以上開いている**

### 3.2 サイボウズ

**公式パッケージの対象業務**:
- 自治体まるごとDXボックス: **予約管理 / 庁内ワークフロー**中心
- ガブキン(650団体・2,900名)のシェアDX優先: **防災・コロナ・広報・公用車日報**
- 災害支援プログラム: 1,000ユーザー無償

**協力隊向け公式テンプレ: ゼロ**

**豊岡事例の公開状況**: cybozu.co.jp/cases/ に**掲載なし**(瀬戸内市・一関市・宮城県庁は掲載あり)

**推測**:
- (a) 1部門にとどまり横展開されていない
- (b) 他自治体への波及効果が乏しい
- (c) 防災・広報・申請などの優先軸と合致しない

**評価: 窓は2-3年。公式パッケージ化の可能性は中程度、ただし時間軸は年単位**

### 3.3 水平SaaS(最大の脅威)

**候補**:
- 楽・楽日報(ESPRIT)
- AI日報(ai-nippo.com)
- gamba!
- Teamspirit
- Leafwork(フィールドレポート)

**脅威度評価**:
- 現在: 自治体調達フロー未対応、協力隊固有フォーマット未対応
- リスク: 「協力隊プラン」追加は参入コスト低い → **2年以内の現実的脅威**

### 3.4 協力隊特化SaaS: **未発見(空白地帯)**

## 4. 自治体調達プロセス

### 意思決定フロー
担当課(協力隊所管・企画系) → 情報政策/DX推進課技術審査 → 財政課査定 → 首長決裁

### 予算サイクル
| 時期 | 動き |
|---|---|
| 7-8月 | 各部局が次年度事業予備検討 |
| **9-10月** | 首長予算編成方針、各部局予算要求書提出 |
| 11-12月 | 財政課ヒアリング・査定 |
| 1月 | 首長査定・復活折衝 |
| 2月 | 議会上程 |
| 2-3月 | 議決、4月執行開始 |

**営業ベストタイミング: 9-10月までに担当課へ情報提供完了**

### 少額随意契約(令和7年4月改正)
- 都道府県・政令市: 工事400万、財産購入300万、**その他(委託・役務)200万**
- **SaaS年額200万円以下なら随契で即調達可能**(基準引上げは追い風)

### 共同調達の前例
| 県 | 対象 | 効果 |
|---|---|---|
| 長野県 | 電子契約(NTT東日本) | 自治振興組合窓口で77市町村が利用 |
| 熊本県 | AI議事録 | 年間コスト3割削減 |
| 三重県 | MIEROPA(データ連携基盤) | みえDXセンター運営 |
| 広島県 | DXShipひろしま | 県・市町DX人材育成 |
