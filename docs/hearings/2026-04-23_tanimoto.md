# 谷本さん(佐用町・企画課室長)ヒアリング 詳細まとめ

## 0. ヒアリングの目的(私側)
**サービスのテスターとして使ってくれるか OK をもらう**こと。

### 結果
- 明確 YES ではないが、**「いいんちゃう」**の温度感
- 有料化はプロポーザル + 金額次第
- **利用者数増 + 事例共有の蓄積**が成立すれば、他社が入っても価格面で多少劣勢でも**使い続けてくれそう**

## 1. 谷本さんプロフィール

- 佐用町出身 → 名古屋の大学
- 大学時代はバックパッカー(タイ・カンボジア・シンガポール・インドネシアを 3 ヶ月)
- 就活せず、母親の助言で佐用町役場に入庁
- 入庁後、ローテーション後に企画課へ
- 企画課で**地域おこし協力隊の管轄を担う年に着任 → 12 年間継続**(創成期からの当事者)
- 現職: 企画課 **室長**
- 入庁後もドイツ・ヨーロッパへ旅行
- **お酒好き**: 白ワインの酸味・渋みが少ないもの、赤ワインも軽め
- **ホタルイカ好き**(炊き込みご飯・パエリア風)

## 2. 佐用町の協力隊の現状

### 体制の変化
- **昨年度: 6 名**
- **今年度: 4 名**(2 名が卒隊)

### 苦悩
- **「人手の補填」**になりがちで、本来の目的(地域づくり)になっていない
- 理想は、**地域づくりに協力する形で協力隊が活動**してくれること
- 現場が「とにかく手が欲しい」と求めるため、調整が難しい

### 佐用町の特産品事情
- **特産品自体はある**が、**伝統工芸品系が薄い**
- ストーリー化・ブランド化に苦戦

## 3. テスター・有料化の道筋

### 谷本さんが語った "OK の条件"
- 当面: **テスターとしての利用は OK**(「いいんちゃう」)
- 有料化したい場合:
    - **プロポーザル(企画提案)**を出す
    - **金額次第**
    - 以下が成立していれば他社が入っても使い続けてくれる:
        - **利用者数が多い**
        - **県全体で使われている**
        - **事例の共有ができている**

### 我々への含意
- **「育てる」と「契約化」は別フェーズ**
- 利用実績 + 事例蓄積 + 県との関係 がそろってから値付け
- 価格競争に対する**ロックイン要素**が必要 = 事例DB(Issue #6)・全国フィード(Issue #18)が直接効く

## 4. 協力隊制度の財源(特別交付税の構造)

### 谷本さんの説明
- 国の**特別交付税**を使用
- 但し性質として**複数用途を含む一括交付**(協力隊専用枠ではない)
- 具体的にどこにいくら充てられているか**不透明**
- 自治体が**申請する全体額より少ない額しか降りない**
- → **雇いすぎリスク**: 想定どおりの交付がないと自治体負担になる
- → 多くの自治体が**人数を増やせない構造的理由**

### 我々への含意
- 「協力隊を増やしたい」自治体に対して、**活動費の透明化(Issue #16)**は強い訴求
- 県全体の集約データがあれば、**交付税申請の根拠資料**として価値
- 補助金提案(Issue #5、Year2)も、財源構造を踏まえた設計が必要
